# Lecture Annotation Tool

강의 중 PDF 슬라이드 위에 실시간으로 주석을 그리는 브라우저 기반 어노테이션 도구.

## 사용법

1. `index.html`을 브라우저에서 열기
2. PDF 파일 드래그&드롭 또는 클릭해서 선택
3. 우측 툴바에서 도구/색상/두께 선택 후 그리기

## 키보드 단축키

| 키 | 동작 |
|----|------|
| `← / →` | 이전 / 다음 슬라이드 |
| `Esc` | 현재 슬라이드 주석 초기화 |
| `1` | 펜 |
| `2` | 화살표 |
| `3` | 동그라미 |
| `4` | 네모 |
| `5` | 텍스트 |
| `6` | 형광펜 |
| `7` | 지우개 |
| `F` | 전체화면 토글 |

## PPT → PDF 변환

- **PowerPoint**: 파일 → 내보내기 → PDF/XPS 문서 만들기
- **Keynote**: 파일 → 내보내기 → PDF
- **Google Slides**: 파일 → 다운로드 → PDF 문서
